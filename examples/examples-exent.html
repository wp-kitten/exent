Beyond JSON. Better than XML.

EXENT (EXtended ENtity Transfer) is the next-generation data transfer object designed for the modern web.

Human Friendly

Comments, optional quotes, and trailing commas make EXENT as readable as a configuration file while maintaining strict data integrity.

Rich Data Types

First-class support for Dates, BigInt, and high-precision Decimals. No more string-parsing hacks for basic data types.

High Performance

Includes a native binary mode (B-EXENT) for rapid machine-to-machine communication with minimal overhead.
