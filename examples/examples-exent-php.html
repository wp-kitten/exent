How it works

1. PHP Backend (backend-exent.php)
header('Content-Type: application/exent');
echo Exent\Exent::stringify($data);
2. Frontend JS
fetch('backend-exent.php').then(response => response.text()).then(text => {
    const data = Exent.parse(text);
    console.log(data.timestamp instanceof Date); // true
});
